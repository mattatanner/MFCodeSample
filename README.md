Running the Project Locally

Clone the repository
Open Visual Studio, select File → Open → Project from Source Control, and enter the repository URL:

https://github.com/mattatanner/MFCodeSample.git


Or use the Git CLI:

git clone https://github.com/mattatanner/MFCodeSample.git


Open the solution
Open BookListCodeSample.sln in Visual Studio.

Restore dependencies
Visual Studio should automatically restore NuGet packages. If not, right-click the solution and select Restore NuGet Packages.

Run the application
Press F5 (or Ctrl+F5) to build and run the Blazor app. The default browser will open to the running application.
